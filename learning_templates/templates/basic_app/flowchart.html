    • Categorical Variable

    • Continuous Variable
      • Intrarater Reliability
        • Single Measurement
          • ICC(1,A)
        • Mean of multiple Measurements
          • ICC(k,A)
      • Interrater Reliability
        • ANOVA Model Assumptions Satisfied
          • Same set of Raters for all Subjects
            • Specific set of rater
              • Single rater
                • Absolute Agreement
                  • ICC(1,A)
                • Consistency
                  • ICC(3,1,C)
              • Multiple raters
                • Absolute Agreement
                  • ICC(k,A)
                • Consistency
                  • ICC(3,k,C)
            • Random set of raters
              • Single rater
                • Absolute Agreement
                  • ICC(2,1,A)
                • Consistency
                  • ICC(1,C)
              • Multiple raters
                • Absolute Agreement
                  • ICC(2,k,A)
                • Consistency
                  • ICC(k,C)
          • Each Subject rated by different set of Raters
            • Single Rater
              • ICC(1,1,A)
            • Multiple Raters
              • ICC(1,k,A)
        • ANOVA Model Assumptions NOT Satisfied
          • Concordance correlation coefficient (CCC)
    • Binomial Variable
      • Fleiss' Kappa
